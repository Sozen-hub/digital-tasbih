#  Digital Tasbih Counter

A simple and responsive Digital Tasbih Counter built using **HTML**, **CSS**, and **JavaScript**. This application allows users to keep track of their dhikr with an easy-to-use interface.

## Features

-  Increment counter
-  Decrement counter
-  Reset counter
-  Responsive design for desktop and mobile devices
-  Clean and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
